## Singleton Design Pattern

- We can have only one object for whole application.
- Comes under creational design pattern

### Benifits

1. Memory Efficiency
2. Data consistency

### How to create Singleton class in java?

1. Private constructore
2. Private static instance
3. Public getter methods
4. Lazy/Eager initialization

### Implementation

- v1: Lazy initialization
- v2: Eager initialization
- v3: Thread-safe
- v4: Double checked locking
- v5: Enum
